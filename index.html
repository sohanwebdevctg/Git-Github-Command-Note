Git & Github command

1) কম্পিউটারে গিট আছে কিনা সেটি চেক করা

=> git -v / git --version

2) কম্পিউটারে গিটহাবের সব কনফিগারেশনের তথ্য

=> git config --list

3) কম্পিউটারে গিটে userName & userEmail set করা

Global:

=> git config --global user.name "username"
=> git config --global user.email "useremail"

Local:

=> git config --local user.name "username"
=> git config --local user.email "useremail"

4) কম্পিউটারে গিটে userName & userEmail আছে কিনা সেটি চেক করা

=> git config user.name
=> git config user.email

5) Git & Github configuration

***Local Repository:

=> git init
=> git add . / git add --all
=> git commit -m "commit message"
=> git status

***Remote Repository:

=> git branch -M main
=> git remote add origin repositoryUlrLink
=> git push -u origin main

***আবার যদি Remote Repository তে ডাটা প্রেরণ করা হয়:

=> git add .
=> git commit -m "commit message"
=> git push

6) গিটে বর্তমান অবস্থা দেখার জন্য

=> git status

7) Commit দেখার জন্য

=> git log / git log --oneline

8) গিটহাব প্রজেক্ট Clone করে আনার জন্য

=> git clone repositoryUlrLink

9) গিটহাব প্রজেক্ট Clone করে আনলেও ফাইল Name Change করা

=> git clone repositoryUlrLink filename

10) গিটহাবে Branch Create করা

=> git branch branchName / git checkout -b branchName

11) গিটহাবে Branch দেখা

=> git branch / git branch --list

12) গিটহাবে Branch পরিবর্তন করতে চাইলে

=> git switch branchName / git checkout branchName (যেই branch এ যেতে চাবো সেই branch নাম)

13) গিটহাবে Branch Name পরিবর্তন করতে চাইলে

=> আগে ওই branch এ থাকতে হবে
=> git branch -m newBranchName

14) গিটহাবে Branch delete করতে চাইলে

=> git branch -d branchName

15) গিটহাবে এক branch এর ডাটা অন্য branch এ আনতে চাইলে (Fetch / Pull)

***Category one:

=> git switch branchName (যেই branch এর ডাটা আনব তার branch যাব এবং branch name তারটা হবে)

=> git merge branchName (যেই branch এ ডাটা আনব সেই branch name হবে)

=> npm update / npm install

***Category two:

=> git pull -u origin branchName (যেই branch এর ডাটা আনব তার branch name হবে)

=> npm update / npm install

16) গিটহাবে Branch এ data push করতে চাইলে

=> git add .
=> git commit -m "commit message"
=> git push -u origin branchName / git push

17) গিটহাব repository পরিবর্তন করে নতুন repository দিতে চাইলে

=> git remote remove origin repositoryUlrLink
=> git remote -v
=> git remote add origin repositoryUlrLink

18) আগের stage এ ফিরত যেতে চাইলে

=> git log / git log --oneline
=> git checkout commitNumberPast (আগের stage এ চলে আসবো)

< ====== Note ====== >

রিপোজিটরি মূলত দুই প্রকার

1)Local

  • Work Directory / Local Directory
  • Staging Area
  • Local Repository

2)Remote

  • Remote Repository(github)

=>Repository: Working Directory ফোল্ডারের সাথে যখন গিট কে যোগ করে দিব তখন তাকে Repository বলে

=>Work Directory / Local Directory: যেই Folder এ প্রজেক্টটি রেখে কাজ করেছি তাকেই Work Directory / Local Directory বলে

=>Staging Area: যেই স্থানে আমরা সবকিছু যাচাই-বাছাই করি, তাকেই Staging Area বলে

=>Local Repository: যখন আমরা working directory তে গিট কে ইন্টিগ্রেট করব সেটা যখন local folder রে বা কম্পিউটারে থাকে তখন সেটি local repository হিসেবে গণ্য হয়

=>Remote Repository: Local Repository কে যখন সার্ভারে আপলোড করব তখন সেটি Remote Repository হিসেবে গণ্য হয়
